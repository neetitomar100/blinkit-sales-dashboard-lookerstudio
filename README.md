# Blinkit Grocery Sales Dashboard

## Project Overview
This project is an interactive sales dashboard created using Google Looker Studio and Google Sheets. The dashboard analyzes Blinkit grocery sales data to identify outlet performance, item category trends, and overall sales insights.

## Tools Used
- Google Looker Studio
- Google Sheets
- Microsoft Excel

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Outlet Performance Analysis
- Item Type Analysis
- Interactive Filters
- Conditional Formatting
- Data Visualization

## KPIs Included
- Total Sales
- Average Sales
- Total Items
- Average Ratings

## Key Insights
- Low Fat products generated higher sales
- Medium outlet sizes contributed the highest revenue
- Tier 1 outlets showed strong sales performance

## Files Included
- Blinkit Grocery Sales Report.pdf
- Blinkit sales dataset.xlsx
